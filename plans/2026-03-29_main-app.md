# Plan: Main App Component

## Date: 2026-03-29

## Objective
Create the main App component that integrates all components and manages global state.

## File Location
`src/App.jsx`

## Component Features

### 1. State Management
```jsx
const [markdown, setMarkdown] = useState('');
const [theme, setTheme] = useState('classic');
const [font, setFont] = useState('Roboto');
```

### 2. Local Storage Persistence (Optional)
```jsx
// Save to localStorage on change
useEffect(() => {
  localStorage.setItem('markdown-content', markdown);
}, [markdown]);

// Load from localStorage on mount
useEffect(() => {
  const saved = localStorage.getItem('markdown-content');
  if (saved) setMarkdown(saved);
}, []);
```

## Component Structure
```jsx
function App() {
  return (
    <div className="app">
      <header>
        <h1>Markdown to PDF</h1>
        <ThemeSelector 
          currentTheme={theme}
          currentFont={font}
          onThemeChange={setTheme}
          onFontChange={setFont}
        />
      </header>
      <main className="app-container">
        <div className="editor-pane">
          <Editor 
            content={markdown} 
            onContentChange={setMarkdown} 
          />
        </div>
        <div className="preview-pane">
          <Preview 
            content={markdown}
            theme={theme}
            fontFamily={font}
          />
        </div>
      </main>
      <footer className="no-print">
        <DownloadButton />
      </footer>
    </div>
  );
}
```
